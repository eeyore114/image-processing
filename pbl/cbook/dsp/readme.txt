■display.exeについて

  display.exeは128×128画素と256×256画素の2バイト整数型および4バイト実数型の
rawデータを画像表示するソフトです。

  ・画像サイズと型の自動判別表示
  ・画像のコピー
  ・画像にグレーバーまたはカラーバーをつけてコピー
  ・表示の最大値、最小値の変更
  ・表示の最大値、最小値を元に戻す
  ・グレーバーとカラーバーの切り替え
  ・画面上の座標とその値の表示
  ・プロファイルの表示（x方向およびy方向）
  ・鳥瞰図の表示

■動作環境（確認済み）

  Microsoft Windows XP の動作する環境
　Microsoft .NET Framework Version 1.0以上が必要です。
　.NET Frameworkにつきましては、Microsoftのホームページを
　ご覧ください。

■インストール

  ○ファイルをコピーすれば、そのまま実行できます。

■アンインストール

  ○display.exeファイルを削除する。


■操作方法

  ［ファイル］メニュー→［開く］で画像を開いてください。

　・「ファイル名」をクリック　→　ファイルを開けます。
　・「Maximum」をクリック 　　→　表示の最大値を変更できます。
　・「Minimum」をクリック 　　→　表示の最小値を変更できます。
　・「バー」の部分をクリック　→　グレーバーとカラーバーを変更できます。
　・「画像部分」を右クリック　→　コピーとバー変更ができます。

　メニューの操作で一通りのことができるようになっています。
